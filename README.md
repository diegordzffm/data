# Data File Examples (CSV, XML, JSON)

This repository contains sample data files I worked with in the context of data processing and analysis.

## Contents
- CSV: transaction datasets for analysis
- XML, SDMX, xbrl: structured reporting data (inspired by financial/regulatory formats)
- JSON: API-like responses

## Tools used
- Python (pandas)
- SQL
- Data validation & transformation

## Purpose
To demonstrate hands-on experience with structured and semi-structured data formats.
